亚马逊北美区 FBA 费用计算页

这是和欧洲版完全独立的新页面，现在统一支持美国、加拿大、墨西哥本地 FBA。 你输入站点、包装尺寸、重量、售价和费率类别后，页面会自动判断 size tier、shipping weight、基础履约费， 并在符合条件时计入各站点对应的 SIPP 折扣、 包装费、low-inventory-level fee 与 overmax handling fee。

  • US 保留 `non-apparel`、`apparel`、`dangerous goods` 三套 2026 非旺季费率。
  • CA 支持 `Envelope / Standard / Oversize` 以及对应的 SIPP discount。
  • MX 支持 3 套费率组、含 VAT 售价带与 SIPP discount。
  • US 会继续计算 `low-inventory-level fee`、`Overmax` 与 bulky `packaging fee`。

输入参数

先选择站点，再输入尺寸、重量和售价。尺寸可用英寸或厘米，重量可用 lb / oz / kg，页面会自动换算成对应站点的计费单位。

美国站按售价带 `< $10` / `$10 - $50` / `> $50` 计费。

当前站点 Size Tier

切换站点后，这里会同步显示该站点的尺寸层级判断标准。

本页只按你提供的北美站 PDF 实现当前规则；欧洲页仍保持独立，不与本页混用。
